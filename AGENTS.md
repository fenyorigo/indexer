# AGENTS

## Project Notes
- Python 3.11+, PyQt6 GUI.
- SQLite via stdlib sqlite3; no ORM.
- ExifTool is an external binary; call with JSON output.

## Common Commands
- Install deps: `python -m pip install -r requirements.txt`
- Run app: `python -m app`
- Tests: `python -m pytest`

## Style
- Python: 4-space indent, Black/Ruff config in `pyproject.toml`.
